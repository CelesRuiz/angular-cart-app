# Angular Cart App

App de carrito de compras desarrollada con Angular 17+.

## Features
- Autenticación con Guard de rutas
- Carrito con persistencia en localStorage
- Formularios reactivos con validaciones
- Signals y componentes standalone

## Tecnologías
- Angular 17+
- TypeScript
- Reactive Forms
- Angular Router

## Cómo correr el proyecto
```bash
npm install
ng serve
```
